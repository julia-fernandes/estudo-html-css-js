# Estudo de fundamentos web básicos - HTML, CSS e JavaScript

Prática proposta em aula
### Exercício 01 - HTML
Estruturação de conteúdo com títulos, parágrafos, imagens e links externos

### Exercício 02 - CSS
Personalização da página com alteração de cores, fonte e formatação de imagem

### Exercício 03 - JavaScript
Implementação de lógica interativa através de funções, caixas de diálogo e botões para manipulação dinâmica do tamanho da imagem.
